## 프로토타입 바로 열기

### 환자 앱
`share-patient.html` 을 브라우저에서 열거나, 아래 Netlify 배포 버전 사용:

```
netlify-deploy/
├── index.html           ← 랜딩 페이지
├── share-patient.html   ← 환자 앱 풀 플로우
├── share-doctor.html    ← 의사 대시보드
└── assets/logo.png
```

## 주요 화면 (10개 스크린)

| 화면 | 설명 |

## 기술 스택

- **React 18** (CDN, no build step)
- **Babel Standalone** (JSX in browser)
- **Pure CSS** (CSS variables + animations)
- 단일 HTML 파일로 배포 가능
