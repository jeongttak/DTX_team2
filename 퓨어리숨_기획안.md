## 2. 문제 정의

### 대상 사용자
- **연령**: 10대 후반 ~ 30대 청년 흡연자 (특히 고·중학생 + 20대 직장인)
- **특성**: 스트레스·사회적 상황·습관으로 인한 반복 금연 실패 경험자

### 기존 치료법의 한계
1. 금연 기간 기록·포인트 부여 수준에 그침 → 충동 순간에 실질적 개입 없음
2. 청년층의 높은 디지털 서비스 수용도를 활용하지 못함
3. 재발 후 다시 시도하도록 돕는 재동기부여 구조 부재

### 근거 기반
- 모바일 기반 실시간 개입이 금연 성공률을 높인다 *(Riley et al., 2008)*
- 게임화 요소는 심리적 관여도와 금연치료 지속력을 촉진한다 *(Schmidt-Kraepelin et al., 2020)*

## 3. 솔루션 개요

> **흡연 욕구가 올라오는 그 순간, 앱이 먼저 손을 내미는 디지털 치료제**

CBT(인지행동치료)와 트리거 분석을 핵심으로,
게임 요소와 캐릭터 성장 시스템을 결합한 금연 DTx.

## 4. 핵심 기능

### 4-1. 온보딩 — 흡연 타입 진단
- 최초 1회, 5점 척도 10문항
- 6가지 흡연 유형 자동 분류 및 비율 산출
- 유형별 맞춤 CBT 코칭 연결

| 유형 | 설명 |
|---|---|
| 스트레스 해소형 | 긴장·짜증을 진정시키려 흡연 |
| 부정적 해석형 | 상황을 자동으로 부정적으로 해석 |
| 자기효능감 부족형 | "이미 망했다 / 못 끊을 거야" |
| 사회적 영향형 | 회식·동료·분위기에 영향 |
| 습관 반복형 | 식후·퇴근 후 자동 흡연 |
| 휴식 확보형 | 쉬는 시간을 확보하기 위한 흡연 |

### 4-2. 트리거 기록 + CBT 소크라테스 질문
- 충동 발생 시 3단계 플로우 자동 진입
  - **1단계**: 충동 강도 + 현재 감정 체크
  - **2단계**: 상황 기록 (장소·동반자·메모)
  - **3단계**: 소크라테스 질문 ("지금 흡연하면 5분 뒤 당신은 어떤 모습일까요?")
- 질문에 답하는 사이 충동(3~5분)이 소멸하는 원리 활용

### 4-3. 호기 CO 측정 연동
- 실제 호기 CO 측정 기기와 연동 (기기 존재 확인 후 기능 추가)
- 매일 금연 인증 → CO 수치 기록
- 미인증 시 3단계 경고 에스컬레이션
  - 1단계 (오후 10시 이전): 노란 경고 배너
  - 2단계 (오후 10시 이후): 빨간 경고 배너
  - 3단계 (자동 실패 처리): CBT 재개입 문구

### 4-4. 게임화 요소
- **금연 스트릭**: 연속 인증일 + 3D 불꽃 카드
- **XP + 코인 보상 시스템**: 미션 완료 시 XP·코인 동시 획득
- **카피바라몬 캐릭터**: 금연 기간·감정 상태 기반 성장형 캐릭터
- **꾸미기 시스템**: 색상·꼬리·뿔·무늬·이목구비·모자 6카테고리
- **도감 수집 시스템**

### 4-5. 미션 시스템
- 일일·주간·월간 3탭 구성
- **자동 연동 미션**: CO 측정, CBT, 트리거 기록 (앱 내 자동 완료)
- **수동 체크 미션**: 물 마시기 (+1잔씩), 산책 (+1바퀴씩), 심호흡
- XP + 코인 보상 / 레벨 시스템

### 4-6. 금연 캘린더
- 월별 인증일(초록) · 흡연일(빨강) · 미측정일(노란 점선) 3분류
  - *미측정 ≠ 흡연* 명확히 구분
- 금연 스트릭 카드 (불꽃 아이콘)
- 날짜 탭 → 해당일 미션 기록 / 회고 메모 바텀시트

### 4-7. 절약 금액 시각화
- 하루 담배값 × 금연일수 자동 계산
- 실생활 환산: "아메리카노 40잔 / 치킨 6마리"

### 4-8. 주간 리포트 *(신규 구현)*
- 요일별 충동 강도 바 차트
- 감정 분포 (스트레스·지루함·불안 등)
- 시간대별 트리거 집중도 히트맵
- 지난주 vs 이번주 비교

### 4-9. 트리거 누적 분석 차트 *(신규 구현)*
- 주별 충동 추이 (4주)
- 충동 유발 상황 TOP 5
- 충동 강도 분포 히스토그램
- CBT 개입 효과 (완료율·참기 성공률)

### 4-10. GPS 위치 기반 트리거 감지 *(시뮬레이션 구현)*
- 흡연 구역 반경 감지 시 자동 알림 배너
- 실제 서비스에서는 Geolocation API 연동 예정

### 4-11. 스마트 알림 — 복합 트리거 분석 *(시뮬레이션 구현)*
- 시간 + 위치 + 감정 3요소 동시 분석
- 고위험 순간 예측 → 충동 발생 **전** 선제적 알림
- 실제 서비스에서는 서버 사이드 AI 분석 + 푸시 알림 연동 예정

### 4-12. MI 동기강화상담 모듈 *(신규 구현)*
- 금연 의지가 흔들릴 때 진입하는 5단계 대화 플로우
  1. 소개 (판단 없는 대화 안내)
  2. 동기 점수 체크 (1~10)
  3. 금연 이유 탐색 (개방형 질문)
  4. 어려운 이유 탐색 (양가감정 인정)
  5. 반영 + 오늘의 다짐 선택
- CBT 소크라테스 질문과 달리 **재발 방지·재동기부여** 초점

## 6. 화면 구성 (12개 스크린)

```
스플래시 (3초) → 로그인 (환자/의사 전환)
  ↓
온보딩 (최초 1회) → 흡연 타입 결과
  ↓
홈 (GPS 감지 배너 · 스트릭 · 절약금액 · 건강지표)
  ├─ 탭바: 미션 / CO 측정 / 캐릭터 / 캘린더
  ├─ 대응 버튼 → 트리거 플로우
  └─ 바로가기: CO 경고 · 스마트 알림 · MI 상담
```
