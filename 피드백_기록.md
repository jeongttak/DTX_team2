# Purely:SOOM 피드백 기록

---

## 📅 2026-05-23 · 1차 발표 피드백

### ✅ 긍정
- 전체적인 앱 구성과 기능 설계가 잘 되어 있음

### 💬 개선 제안

**1. 하드웨어 기기 연동 계획 명확화**
- 호기 CO 측정 기기를 앱과 어떤 방식으로 연동할지 설명 필요
- 블루투스(BLE) 기반 연동 여부 포함한 기술 방향 정리 요청

**2. 의료 데이터 수집 동의 팝업**
- 의료 서비스 특성상 앱 진입 시 개인정보·데이터 수집 동의 절차 필요
- 프로토타입에도 구현 시 신뢰성 향상 효과
- → 구현 예정

**3. 블루투스(BLE) 연동 방식 확인**
- 기기 연동을 블루투스로 할 것인지 확인 요청

**4. 호기 CO 측정 화면 UI 개선**
- 숨 부는 측정 화면을 시각적으로 더 아름답게 개선 필요
- 참고: 삼성 안드로이드 헬스 앱의 호흡 UI
- 꽃잎 날리는 애니메이션 방향으로 개선 요청
- → 구현 예정

**5. 디자인 통일성**
- 블러 효과 과다 사용 → UI가 복잡해 보임, 최소화 권장
- border-radius 요소별 기준 통일 필요
  - 어떤 건 너무 사각형, 어떤 건 너무 원형
  - 카드·버튼·칩 등 요소별 고정값으로 통일
- → 구현 예정
